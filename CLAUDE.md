## Design System

Always read `DESIGN.md` before making any visual or UI decisions.

All font choices, colors, spacing, layout, and motion guidance are defined there.
Do not introduce new UI tokens or visual patterns unless they extend `DESIGN.md` intentionally.

When touching existing UI, prefer converging toward the shared design system over adding one-off styles.
